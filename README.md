# Prompt Collector Extension

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Synthetic-Alchemist/prompt-collector-extension.git
   ```
2. Navigate to the project directory:
   ```bash
   cd prompt-collector-extension
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
1. Start the application:
   ```bash
   npm start
   ```
2. Open your browser and go to `http://localhost:3000` to use the extension.

3. Follow the prompts to collect and manage your data effectively!
